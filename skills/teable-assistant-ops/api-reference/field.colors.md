## Color Variants

Each color family includes variants:
- **Light2**: Lightest shade
- **Light1**: Light shade  
- **Bright**: Bright/saturated (recommended for visibility)
- **Base**: Standard color (e.g., "blue", "green")
- **Dark1**: Darker shade
